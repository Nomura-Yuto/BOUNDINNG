# 【概要】
- **制作環境：** OpenGL / C++ / C  
- **制作人数：** 1人  
- **制作期間：** 3ヵ月 (2023/06 ～ 2023/08)  


# 【ゲーム詳細】
Switch上で動くゲームです。  
上から落ちてくるブロックに当たらないようにJoy-Conを傾けて避けながら、  
Joy-Conをタイミングよく振りスコアを稼いでいくゲーム。
